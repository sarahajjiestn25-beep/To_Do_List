# 📝 To-Do List App

A simple and interactive **To-Do List application** built in Python. This project allows users to manage daily tasks easily through a command-line interface. Perfect for beginners to practice Python basics, loops, conditionals, and list management.

---

## ✨ Features

- 🟢 **Add tasks**: Easily add tasks to your to-do list.  
- 📋 **View tasks**: Display all your current tasks with numbering.  
- ❌ **Delete tasks**: Remove completed or unwanted tasks by selecting the task number.  
- 🖐️ **Exit safely**: Close the app gracefully with a goodbye message.  

---

## 🚀 How to Run

1. Make sure you have **Python 3** installed.  
2. Clone this repository or download the `to_do_list.py` file.  
3. Open a terminal or command prompt in the project directory.  
4. Run the app with:

```bash
python to_do_list.py
```

5. Follow the on-screen instructions to **add**, **view**, or **delete** tasks.
